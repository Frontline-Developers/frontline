# Frontline — Flutter App

This is the Flutter application for Frontline. See the [root README](../../README.md) for full setup instructions, project structure, and development guide.

## Quick start

```bash
# From repo root
./setup.sh      # first-time setup
./dev.sh --web  # run on Chrome with local emulators
./dev.sh        # run on Android with local emulators
```

cd apps/mobile
